# 🌱 SmartHorta

Sistema de Monitoramento e Controle de Irrigação

## 📌 Sobre o Projeto

O SmartHorta é um sistema desenvolvido com o objetivo de auxiliar no monitoramento da umidade do solo e no controle da irrigação de hortas, utilizando conceitos de tecnologia e sustentabilidade.

O sistema permite registrar medições de umidade, analisar os dados e fornecer recomendações automáticas sobre a necessidade de irrigação.

---

## 🚀 Funcionalidades

- Registro de umidade do solo
- Classificação automática:
  - 🔴 Seco (regar)
  - 🟡 Ideal
  - 🟢 Úmido (não regar)
- Medições 
- Armazenamento local (LocalStorage)
- Gráfico de evolução da umidade
- Interface simples e intuitiva

---

## 🧠 Regras de Negócio

- Umidade < 30% → 🔴 Regar
- Umidade entre 30% e 60% → 🟡 Ideal
- Umidade > 60% → 🟢 Não regar
